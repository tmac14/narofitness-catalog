Conjunto de 5 alternativas en SVG, pensadas como **app mark simplificado 32x32 px** para top bar o app chrome.

1. `01_hexagon_catalog_nr.svg` — monograma NR en marco hexagonal con guiño de catálogo/grid.
2. `02_dumbbell_shield_nr.svg` — escudo técnico con guiño de mancuerna/equipamiento.
3. `03_tech_tiles_nr.svg` — app mark con marco rounded-square, corner fold y tiles de catálogo.
4. `04_kettlebell_nr.svg` — silueta kettlebell con NR legible.
5. `05_plate_ring_catalog_nr.svg` — ring/plate con núcleo técnico y guiño de catálogo.
